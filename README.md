# Our Projects
<a href="https://github.com/Linta-Inc/Nexusify">Nexusify</a>


The project Nexusify aims mid-range devices, such as Galaxy A15 4G, 5G, Galaxy A16 4G, 5G and more. 

The official supported device is only Galaxy A15 4G and 5G. The model numbers can be found in the repository.

### Highlight features of Nexusify:
- Galaxy AI Support
- Live Blur Support
- Spoof One UI Version to 8.0
- S25 Wallpapers & Sounds
- Smoother animations
- 4k@60fps Video Recording Support

The features are not limited to this list. Check out repository for all features.
